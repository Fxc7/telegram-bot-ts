# Telegram Bot TS

## Instalation

### with npm installed

```bash
npm install --save-dev typescript && npm install && npm start
```

### with pnpm installed

```bash
pnpm install --save-dev typescript && pnpm install && pnpm start
```

### with yarn installed

```bash
yarn add --save-dev typescript && yarn install && yarn run start
```
